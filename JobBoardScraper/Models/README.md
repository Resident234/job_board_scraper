# Models

Папка содержит модели данных (Data Transfer Objects) для JobBoardScraper.

## Структуры данных

### CompanyDetailsData
Детальная информация о компании, извлекаемая со страницы компании.

### UserProfileData
Данные профиля пользователя (уровень, зарплата, опыт, публичность).

### ResumeProfileData
Данные профиля со страницы навыков (для быстрого парсинга списков).

### UserExperienceData
Информация об опыте работы пользователя (компания, должность, навыки).

## Вспомогательные классы

### ScraperStatistics
Статистика работы скрапера (обработано, успешно, ошибок, время).

### HttpRequestResult
Результат HTTP запроса с метаданными (статус, время, ошибки).

## Использование

Все модели используются для:
- Типизированной передачи данных между компонентами
- Валидации структуры данных
- Улучшения читаемости кода
- Автодополнения в IDE
